# Sensor-Fault-detection
传感器故障检测，多输入输出神经网络模型

传感器数据使用仿真数据，共有7个输入和4个输出，分别是：
    CS_angle：操纵面偏转量（机翼）单位：度;
    F_alt：飞行高度 单位：米
    rate_HC：高度变化速率 单位：米/秒
    P_angle：俯仰角 单位：度
    Windspeed：风速 单位：米/秒
    Airspeed：空速 单位：节
    Engine_thrust：引擎推力 单位：千克
    Attack_angle：迎角 单位：度
    P_rate：俯仰角速率 单位：度/秒
    Roll_rate：滚转角速率 单位：度/秒
    Yaw_rate：偏航角速率 单位：度/秒
